# Book Explorer Web Application

A simple and clean **Book Explorer** web application built using **React and TypeScript**.  
The application allows users to log in, browse a list of books, search by title or author, and view detailed information about each book.

This project was developed as part of a technical assignment.

---

## 🚀 Features

- Login with form validation
- Mock authentication using LocalStorage
- Protected routes (Book List & Book Details)
- Search books by title or author
- View detailed information for each book
- Graceful handling of invalid book IDs
- Optional loading indicator for better user experience
- Clean and minimal UI

---

## 🛠 Tech Stack

- React
- TypeScript
- React Router
- LocalStorage
- Mock local data
